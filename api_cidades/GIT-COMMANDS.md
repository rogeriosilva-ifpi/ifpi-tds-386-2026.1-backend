## No terminal, dentro da pasta
- ```git init``` --> inicilizar a pasta como repositório git
- ```git add .``` --> Adicionar todos os arquivos para o próximo commit
- ```git commit -m "mensagem "``` --> Confirma as mudanças
- ```git status``` --> Repositório tá 100%


## No Github, na internet
- Criar uma conta se não tiver
- Criar um repositório (público ou privado)
- Ele mostrar a Orientações de como adicionar um Remoto (linkar com seu local)
- Copiar ```git remote add ...``` (tipo HTTPS)
- ```git push``` --> perdir usuário (email) e senha (token descrito abaixo)
- Antes vc vai criar um Personal Access Token (para servir de senhar no PUSH):
- Vá na foto perfil > (Avatar) > settings > Developer Settings > Personal Access Token (classic)
- Esse token acima será sua senha ao fazer PUSH
